Q: Can I use transparency in textures?
A: Yes! Use PNG format with alpha channel.

Q: How many textures can I load?
A: Limited by GPU memory. Typically 100+ textures of 1024x1024.

Q: Can I have different textures on different walls?
A: Yes! Apply one texture, then replace the file and apply to next wall.

Q: What if I want to use textures from a specific folder?
A: Edit test.cpp line ~1296 and add custom paths to texturePaths[] array.

Q: Can I rotate or scale textures?
A: Not currently. UV coordinates are auto-generated based on geometry.

Q: Does OBJ export include textures?
A: Not yet. This is a planned feature for future updates.

Q: Can I see texture coordinates?
A: They're auto-calculated. Check DrawWall() function for UV mapping logic.
