┌────────────────────────────────────────────────────────────────┐
│ HOW TO ADD TEXTURES (4 SIMPLE STEPS)                          │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  STEP 1: CREATE A WALL                                        │
│  ├─ Press [1] for SELECT MODE                                 │
│  ├─ Click 3+ nodes (must be coplanar)                         │
│  └─ Press [SPACE] to create wall                              │
│                                                                │
│  STEP 2: PREPARE TEXTURE FILE                                 │
│  ├─ Get a PNG or JPG image                                    │
│  ├─ Rename to: texture.png (or wall.png, tex.png)            │
│  └─ Place in: C:\Users\Kris\raylib_project\                  │
│                                                                │
│  STEP 3: APPLY TEXTURE                                        │
│  ├─ Hover mouse over the wall                                 │
│  └─ Press [T] key                                             │
│                                                                │
│  STEP 4: DONE!                                                │
│  └─ Texture is now applied to the wall                        │
│                                                                │
└────────────────────────────────────────────────────────────────┘

│ SUPPORTED IMAGE FORMATS                                        │

┌────────────────────────────────────────────────────────────────┐
│ RECOMMENDED TEXTURE SIZES                                      │
├────────────────────────────────────────────────────────────────┤
│  Small walls:    512x512 pixels                               │
│  Medium walls:   1024x1024 pixels  ← RECOMMENDED              │
│  Large walls:    2048x2048 pixels                             │
│  Maximum:        4096x4096 pixels (GPU dependent)             │
│                                                                │
│  TIP: Use power-of-2 dimensions for best performance!         │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  PROBLEM: "No texture file found"                             │
│  ├─ Check file is in project root folder                      │
│  ├─ Verify filename matches (texture.png, wall.png, etc.)    │
│  └─ Check file extension is correct                           │
│                                                                │
│  PROBLEM: Texture appears stretched                           │
│  ├─ Use rectangular walls for best results                    │
│  ├─ Keep all wall nodes coplanar                              │
│  └─ Use simpler wall shapes (3-4 nodes)                       │
│                                                                │
│  PROBLEM: Texture is blurry                                   │
│  ├─ Use higher resolution texture                             │
│  └─ Recommended: 1024x1024 or 2048x2048 pixels               │
│                                                                │
│  PROBLEM: "T key pressed but no wall hovered"                │
│  ├─ Make sure mouse is over a wall                            │
│  ├─ Wall should highlight when hovered                        │
│  └─ Try moving mouse slightly                                 │
│                                                                │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ TIPS & BEST PRACTICES                                          │
├────────────────────────────────────────────────────────────────┤
│  ✓ Use seamless/tileable textures for best results            │
│  ✓ PNG format for quality, JPG for smaller file size          │
│  ✓ Keep textures under 2048x2048 for good performance         │
│  ✓ Reuse textures - apply same texture to multiple walls      │
│  ✓ Test texture on small wall first                           │
│                                                                │
│  ✗ Avoid mixing different texture styles                      │
│  ✗ Don't use huge 4K textures unless necessary                │
│  ✗ Avoid complex wall shapes (5+ nodes)                       │
